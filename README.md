# 🛒 Java Shopping Cart System

A simple console-based Shopping Cart application developed using Java. This project demonstrates the use of variables, user input, conditional statements, arithmetic operations, and basic Java programming concepts.

## 🚀 Features

- 🛍️ Accepts customer name
- 📦 Allows users to enter product names
- 💰 Displays product prices
- 🧮 Calculates total bill
- 📦 Adds packaging charges
- 🎁 Checks discount eligibility
- 🚚 Displays delivery information
- 💻 Console-based Java application

## 🛠️ Technologies Used

- Java
- VS Code
- Scanner Class

## 📂 Project Structure

```
Shopping.java
README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Open the project in VS Code or any Java IDE.
3. Compile the program:
   ```
   javac Shopping.java
   ```
4. Run the program:
   ```
   java Shopping
   ```
